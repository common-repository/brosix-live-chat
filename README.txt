== Installation ==

= Automated Installation =

1. Go to your WordPress admin dashboard, then find the “Plugins” menu.
2. Click on “Add New” and search for “Brosix Live Chat”.
3. Activate the “Brosix Live Chat” plugin through the `Plugins` menu in WordPress.
4. Click the `Live Chat` menu on the left.
5. Create a new account or sign-in if you already have a Brosix account.

= Manual Installation = 

1. Upload `brosix-live-chat` directory to WordPress plugins directory (`/wp-content/plugins/`).
2. Activate the “Brosix Live Chat” plugin through the `Plugins` menu in WordPress.
3. Click the `Live Chat` menu on the left.
4. Create a new account or sign-in if you already have a Brosix account.

For more detailed instructions, go to the [live chat page](https://www.brosix.com/live-chat/?utm_source=wordpress.org&utm_medium=integrations&utm_campaign=wordpress-integration).

== Frequently Asked Questions ==

= How much does Brosix cost? =

Brosix is FREE for all Wordpress users to install and run Live Chat on their website and support their customers with one agent. The agent can use Brosix apps to respond to website visitors on all major platforms - Windows, Mac, Linux, Android, iOS and Web.

For additional agents you can upgrade to the paid Premium plan, which also provides a wider range of collaboration and administration features, that will make your team more productive.

= Is the Live Chat widget customizable? =

Yes it is. You can set different levels of availability for Live Chat widget: always available, available when an agent is online, and available only during set business hours. You can also adjust the placement of the widget on the left or right sides of your webpage, and set a color of your choice for the Live Chat icon. 

= Can I use Brosix on more than one website? =

Yes, there are two options to use Brosix on multiple websites. You can use the same chat on every website or you can create different projects for each website, allowing you to manage each one separately.

= Is there a mobile application? =

Yes, there are both iOS and Android mobile apps. You can download them from the App Store or Google Play.
